# Shifts Organizer
Python script which organizes shifts based on variable criteria

# Config

## Shift 1
- List of people names
  - Name 1
  - Name 2
- Time zone - Default: EET
- Day shift start - Default 09:00
- Day shift end - Default 18:00
- Night shift start - Default 17:00
- Night shift end - Default 02:00
- Wednesday day shift overfill count - Default: 2
- Prefer 2 or more people in shift - Default: true

## Shift 2
- List of people names
  - Name 1
  - Name 2
- Time zone - Default: GMT+7
- Day shift start - Default 09:00
- Day shift end - Default 18:00
- Night shift start - Default 17:00
- Night shift end - Default 02:00
- Wednesday day shift overfill count - Default: 2
- Prefer 2 or more people in shift - Default: true

## Extra
- Maximum day shifts in a row - Default: 5
- Maximum night shifts in a row - Default: 5
- Maximum days off - Default: 2
- Minimum days off - Default: 1
- No day shift after night shift - Default: true
- Friday shift 2 priority names - Default: ""
- Wednesday day shift overfill - Default: true
